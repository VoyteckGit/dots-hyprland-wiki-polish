# It is recommended to switch to the new Quickshell version
- Much better performance, at the cost of losing some niche customizations and higher memory usage

# Here's what to expect
- A practical, accessible environment built for everyday use
- A user-friendly interface that doesn't require you to go down the rabbit hole
- Keybinds are similar to that of Windows

---

- _Note: Dotfiles are heavily opinionated software._

# Zalecane jest przejście na nową wersję opartą na Quickshell
- Znacznie lepsza wydajność kosztem utraty niektórych niszowych personalizacji oraz większego użycia pamięci

# Czego się spodziewać
- Praktyczne, dostępne środowisko stworzone do codziennego użytku
- Przyjazny interfejs, który nie wymaga gimnastykowania się
- Skróty klawiszowe podobne do tych z systemu Windows
